lsdn_err_t lsdn_virt_get_recommended_mtu ( struct lsdn_virt *  virt,
unsigned int *  mtu 
)

Get a recommanded MTU for a given virt.

The MTU is based on the current state and connection port of the virt (it is not based on the committed stated). The phys interface must already exist.

Get a recommanded MTU for a given virt.

Calculates the appropriate MTU value, taking into account the network's tunneling method overhead.
