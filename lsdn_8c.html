static void abort_handler ( const struct lsdn_problem *  problem,
void *  user 
)
static

Problem handler that aborts when a problem is found.

Used in lsdn_context_free. When freeing a context, we can't handle errors meaningfully and we don't expect any errors to happen anyway. Any reported problem will then simply dump description to stderr and abort.
