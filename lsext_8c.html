static

Get a net argument, either explicit or from scope.

Find out if the command is either nested inside a net {} scope or if it has explicit -net keyword argument.

The caller is still responsible for parsing the keyword arguments using Tcl_ParseArgsObjv and only passes in the extracted value for -net.

Parameters
argThe -net keyword argument value.
netOutput value. Undefined if TCL_ERROR is returned. May be null if needed is false.
neededCause a TCL_ERROR if neither the -net argument is given or scope is active.
Returns
Tcl status.
