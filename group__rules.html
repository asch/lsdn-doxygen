◆ LSDN_MAX_MATCH_LEN

#define LSDN_MAX_MATCH_LEN   16

Maximum size of lsdn_matchdata struct, in bytes.

This should be the same as sizeof(union lsdn_matchdata). But this value is used to declare size of lsdn_matchdata.bytes field, so using sizeof is not possible.
