bool lsdn_phys_attachment::explicitly_attached

Was this attachment created by lsdn_phys_attach at some point, or was it implicitely created by lsdn_virt_connect, just for bookkeeping? All fields below are valid only for explicit attachments.

If you try to commit a network and some attachment is not explicitly attached, we assume you just made a mistake.
