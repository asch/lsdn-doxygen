◆ ack_decommit()

static bool ack_decommit ( enum lsdn_state *  s)
inlinestatic

Help process object deletion or renew.

If the object was in LSDN_STATE_DELETE, the state will not change (the whole object will be deleted later instead, use ack_delete for that). If the object was in LSDN_STATE_RENEW, it must be reinstalled, so the state is set to LSDN_STATE_NEW. Does nothing for other states.

Parameters
sPointer to commit state.
Return values
trueThe caller should decommit the object, because it is getting renewed or deleted.
falseThe object was in other state.

◆ ack_state()

static void ack_state ( enum lsdn_state *  s)
inlinestatic

Mark object as commited.

If the state was LSDN_STATE_NEW or LSDN_STATE_RENEW, that means the object is now commited. The new state is LSDN_STATE_OK.

If the state was already OK, no change. If the state was LSDN_STATE_DELETE, the object is now deleted, so LSDN_STATE_DELETE is the only possible appropriate value.

Parameters
sPointer to commit state.
