◆ lsdn_name_set()

lsdn_err_t lsdn_name_set ( struct lsdn_name *  name,
struct lsdn_names *  table,
const char *  str 
)

Update an existing name.

If the new name is the same as the old name, returns LSDNE_OK immediately. Otherwise checks for uniqueness within table and if that succeeds, assigns the new name to the name variable.

Makes a private copy of str, so that str can be freed or reused after calling. This is useful because str is usually returned from lsdn_mk_name, which reuses a fixed buffer in the current context.

Parameters
nameName struct.
[in]tableList of names.
[in]strNew name.
Return values
LSDNE_OKif the update was successful.
LSDNE_DUPLICATEif the name already exists in table.
LSDNE_NOMEMif memory allocation failed.
